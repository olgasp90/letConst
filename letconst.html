Quiz

  • What is the difference between var and let?

    var is function scoped and let is block scoped. A variable declared with var can be reassigned and redeclared, while a variable declared with let can be reassigned, but never redeclared

  • What is the difference between var and const?

    var is function scoped and const is block scoped. A variable declared with var can be reassigned and redeclared, while a variable declared with const cannot be reassigned and connot be redeclared

  • What is the difference between let and const?

    Both are block scoped, but a variable declared with let can be reassigned, while the value of a variable declared with const cannot be reassigned

  • What is hoisting?

    Hoisting is JavaScript's default behavior of 'moving declarations to the top'. That means that a variable declared with var that exists in the document can be 'seen' that it exists before we actually see it's value and if we console.log that variable we will get undefined
